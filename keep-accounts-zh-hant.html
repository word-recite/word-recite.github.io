使用說明 (v1.09)

 简体中文 English

1. 賬目分類和子分類

系統將賬目記錄劃分爲：分類 -> 子分類 -> 記錄 三級管理。每條記錄隸屬於一個子分類，每個子分類隸屬於一個分類。 您可以進入"分類"標籤頁對分類進行管理。 首次使用時,可以點擊"導入內置分類列表"，系統會自動添加一組分類和子分類。您可以自行調整這些分類以適應您的需求。

注意：

  • 添加子分類時，需要選擇「收入」或「支出」類型。收入子分類中的記錄全部爲收入記錄，支出子分類同理。
  • 長按分類或子分類名稱，在彈出菜單中可以選擇 "編輯"或"刪除"分類或子分類。注意刪除後，其下的子分類和記錄都會被刪除。

2. 記賬

進入"分類"標籤頁,點擊一個子分類或進入"匯總"標籤頁，點擊"添加記錄"按鈕。 在彈出的窗口中填入相應信息後點擊右上角的"添加"。有如下信息可以輸入：

字段注意
金額 點擊金額右側的按鈕可以選擇其他貨幣，系統會自動換算成當前貨幣對應金額。
可以輸入+，-，*進行簡單運算。
商家 點擊商家右側的按鈕可以顯示之前在相同子分類下記錄過的商家列表。
標籤 用在"設置">"高級"頁中定義的標籤標記記錄，之後可以在「圖表」頁查看不同標籤下的數據匯總。
日期 點擊右側箭頭，在展開的內容中填入信息。
詳情 點擊右側的按鈕後填入信息。
連續添加 選中後點擊"添加"不會關閉窗口，以便添加下一筆記錄。添加分類、標籤、自動記錄模版時也可以同樣操作。

如果有賬目相關的圖片可以記錄（如拍下消費收據或發票的圖片），可以點擊"添加圖片記錄"。 並點擊彈出界面中的"拍照"或"圖庫"。也可以直接將圖片從其他APP拖放到添加對話框的圖片區域。

用手指在圖片上滑動會顯示一個綠色邊框的方塊， 點擊下方的"剪切"按鈕可以剪切並放大方塊內的圖像。

點擊「分析」，系統會識別綠色邊框內的圖片（未選擇則分析整個圖片）中包含的金額、日期、商家等信息並提示用戶選擇填入,並自動推薦三個分類供選擇。

注意：

  • 商家信息必須在之前的記錄中出現過，且在圖片文字中出現才可以自動識別。
  • 建議選擇圖片中的相關區域後再分析，以提高準確度。
  • 「分析」中的分類推薦是根據以往記錄中的詳情、商家、附帶圖片中的文字進行統計得到，如果這些信息比較少，則推薦可能不準確
  • 如果系統中帶圖片的記錄較多，首次使用「分析」時可能分類推薦爲空，這是系統在分析現有記錄，過一段時間後再分析，分類推薦即可工作。

3. 匯總

進入"分類"標籤頁並進入某個子分類，或進入"匯總"標籤頁可以查看每一天的收支記錄，以及當月和當年的總收入和總支出。 可以點擊上方的日期並在彈出菜單中選擇不同的日期進行匯總。

注意：

  • 日期彈出菜單只會包含有記錄的日期。
  • 可以選擇菜單中的"全部",以顯示整年（僅在分類標籤頁有效）或整月的記錄。
  • 點擊記錄以顯示更多信息，可以點擊圖片以放大顯示。
  • 長按記錄，在彈出菜單中可以選擇 "編輯"或"刪除"記錄。
  • 可以移動記錄到其他子分類。方法是點擊右上角，並選擇「開始選擇」。 選好要移動的記錄後，進入目標子分類，再點擊右上角，選擇「移動至此」。 完成後點擊剛才彈出菜單中的「完成選擇」，結束選擇狀態。
  • 進入"分類"標籤頁,點擊"數據回收"。這裡列出了已經刪除但還沒有用iCloud同步的記錄， 可以點擊右上角"編輯"，選擇並移動記錄到其他子分類，就可以還原被刪除記錄。
  • 進入"分類"標籤頁,點擊「查找記錄」。可以在這裡用關鍵字在所有記錄的「詳情」和「商家」信息中查找。

4. 圖表

進入"圖表"標籤頁可以按記錄的不同字段使用圖表進行匯總統計。如按年匯總統計，每一年再按月進行匯總統計等等。 點擊「匯總方式和順序」右側的"重置"會恢復到如下缺省匯總字段和順序：
  • 按收支匯總（此字段必須出現且不能改變順序）
  • 按標籤匯總
  • 按年匯總
  • 按月匯總
  • 按日匯總
  • 按分類匯總
  • 按子分類匯總
  • 按商家匯總
點擊「匯總方式和順序」右側的"編輯"，可以刪減字段，並調整順序。

5. 設置

點擊應用下方的「設置」，可以看到以下按鈕和功能

升級軟件
(無限制使用） 		未購買本軟件之前只能添加50條記錄，且不能使用"從文件導入"功能。 點擊"以XX價格購買本軟件"， 成功購買後會顯示「已購買」。

注意:
購買之後如果重新安裝了本應用或在其他設備安裝了本應用，需要點擊"恢復購買"。
同步iCloud 同步分類和記錄數據到iCloud。 同步時會自動檢測本地的更改並上傳到iCloud，同時下載iCloud上的更新到本地。 如果本地和iCloud上數據在相同字段都有更新，則以本地更新優先。
自動同步 選中後，系統會每隔2分鐘自動同步iCloud。
進行身份驗證 選中後，每次進入應用後會使用FaceID、TouchID或密碼驗證身份。
導出到文件 導出分類和記錄數據到CSV文件。注意：不會導出記錄中的圖像數據。
從文件導入 從CSV文件導入分類和記錄數據，導入時會忽略CSV文件中本地已存在的記錄。
自動添加模版 添加每月自動記錄（如每月31日支出iCloud服務費X元）。 如果當月不含有此日期，則自動記錄在最後一日（如2月不含31日，則自動記錄在最後的28或29日）。
標籤管理 添加自定義標籤。添加後，在添加記錄時就可以選擇這個標籤來標記記錄。

6. 導入/導出文件格式

導入/導出CSV文件中每一行包含一個分類、子分類、標籤或記錄的數據,各字段用逗號分隔。

分類包含以下字段：
  1. 0
  2. 分類ID，任意可唯一標識分類的字符串
  3. 分類名稱
子分類包含以下字段：
  1. 1
  2. 子分類ID，任意可唯一標識分類的字符串
  3. 子分類名稱
  4. 上級分類ID
  5. true -- 收入子分類， false -- 支出子分類
記錄包含以下字段：
  1. 2
  2. 記錄ID，任意可唯一標識記錄的字符串
  3. 上級子分類ID
  4. 記錄日期(年-月-日 時:分)，如2022-01-12 05:15
  5. 金額， 如100.00
  6. 0
  7. 商家
  8. 詳情
  9. 標籤ID
標籤包含以下字段：
  1. 3
  2. 標籤ID，任意可唯一標識標籤的字符串
  3. 標籤名稱
  4. 標籤顏色：顏色的RGB數值，範圍在0-1之間，用|分割， 如"0.5|0|0.5"
