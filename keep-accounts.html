1. Category and Subcategory

This application groups account records into subcategories, and group subcategories into categores. After click "Import built-in category list" in "Category" tab, a set of categories and subcategories will be generated. You can add/delete/modify these categories and subcategories for your requirement in "Category" tab.

Notice:

  • When adding subcategory, the subcategory can be created as an incoming subcategory which only contains incoming records, or an expense subcategory which only contains expense records.
  • To edit/delete a category/subcategory, long press the name of the category, select 'Edit' or 'Delete' in the popup menu.

2. Book keeping

Enter 'Category' tab, click one subcategory or enter 'Summary' tab, then click "Add record" button, fill information of the new record, click "Add" in the upper right corner.

FieldsNotes
Amount you can choose different currency by clicking the button next to amount input field.
You can input +,-,* to do simple calulation in amount input field.
Merchant You can select from merchants previously recorded in same subcategory by clicking button next to merchant input field.
Label Mark record with a label defined in "Settings"/"Advanced" page. The summary of different labels can be viewed at "Chart" tab.
Date Click right arrow button, fill information in expanded view.
Detail Click on the right and fill the information。
Continuous add The dialog will not be closed after clicking "Add"，so that you can add next record. This is also supported in label, category and auto-insert template adding dailog。

By clicking "Add record with image", you can add record with picture from the Photos or Camera(For example, add an image of receipt or invoice). You can also add an image by dragging it from other app to the image area.

You can use your finger to drag a green rectangle on the image to select a region of image and click "Crop" to crop the region out of image.

Click "Analyse", list of money amount, date, merchants will be prompt in a new window if there is any contained in selected image region(or the whole image if no selection). Three categories will also be prompt for selection.

Notice：

  • The merchants information can only be recongnized when there is at least one previous record contains the information.
  • It is better to select relavent region before click "Analyse" to improve accuracy
  • The suggested categories in “Analyse” is based on existing record's details, merchants and text in attached images. If there are too few of these information, the suggestion may not be accurate.
  • If there are many existing records attached with image，“Analyse”may not give suggestion on categories in the first time of use. this is because system is busy with analyzing current records，it may work after a while.

3. Summary

Enter 'Category' tab, click one subcategory or enter 'Summary' tab, here you can view all the records in a particular day. The total expense and income of the whole year and month will also be shown. Click the year, month and day text on the top will pop up menu to switch displaying date.

Notice:

  • The popup menu for switching date will only contain date which has records.
  • You can select "All" to show records of the whole year(only valid in Category tab) or month.
  • You can click the record to view more information, you can click the image if any to enlarge it.
  • You can edit/delete record by long pressing the record.
  • You can move the records to different subcategory. First click in the upper-right corner and select "Start Selection" in the popup menu. Then select records which need to be moved. Enter the target subcategory page, click in the upper-right corner and select "Move to here". Click "Done selection" in the same popup menu to finish selection state.
  • In "Category" tab, click "Data Recycle". Here, you can view records deleted but not synchronized with iCloud. Click "Edit" button in the top-right corner, select and move deleted records to other subcategory to restore them.
  • In "Category" tab, click "Search records". Here, you can use keyword to search in all records's detail and merchant information.

4. Chart

In "Chart" tab, you can view statistic on the records by different angle. For example, show total sum of the records for each year, then in each year, show total sum of the records for each month. By click "Reset" in the upper right corner of "Summary fields and order" section, the following fields and order will be set for histogram:
  • Balance Summary(This field can not be deleted and moved)
  • Summary By Label
  • Yearly Summary
  • Monthly Summary
  • Daily Summary
  • Summary By Category
  • Summary By Subcategory
  • Summary By Merchant
By click "Edit" in the upper right corner of "Summary fields and order" section, you can delete or reorder these fields.

5. Settings

Enter the “Settings” tab, here you can use following functions and settings.

Upgrade App
(unrestricted use) 		Before paying for upgrading the app, you can only add 50 records and "Import from file" function can not be used. Click"Upgrade App at price XXX" to pay and upgrade the app. After upgrade, text “Purchased” will replace the upgrade button.

Notice:
If you reinstalled the app, or installed the app on another device after upgrading, you can click "Restore Purchase" to restore purchase state.
Sync iCloud Sync categories, subcategories and records to iCloud. The local updates will be uploaded to iCloud, and the remote changes will be downloaded to local device. Note that if you have updates on same field of the same category or subcategory or record from both local and iCloud, the local update will overwrite iCloud update.
Auto-Sync Automatically synchronize iCloud every 2 minutes.
Owner authenticate Authenticate using FaceID, TouchID or Password after entering the app to protected your privacy.
Export to file Export categories, subcategories and records data to CSV file. Note that image data in records will be ignored.
Import from file Import categories, subcategories and records data from CSV file. Note that only data in CSV file that does not exists in local device will be imported.
Auto-insert template You can config the app to add record automatically in a specific day of each month (For example, add X dollar expense at day 31 of each month for iCloud monthly service fee). If a date is not valid in certain month, the auto-created record will be set for the last day of the month. (February does not have day 31, so the new record date will be set to 29 February or 28 February)
Label Management Add self defined label, you can mark each record with one of labels you defined. The summary of different labels can be viewed at "Chart" tab.

6. Import/Export cvs file format

Each line in imported/exported cvs file contains data of one category, subcategory, label or record, seperated by comma.

Category contains following fields:
  1. 0
  2. Category id, any string that can uniquely identify the category.
  3. Category name
Subcategory contains following fields:
  1. 1
  2. Subcategory id, any string that can uniquely identify the subcategory.
  3. Subcategory name
  4. Parent category id
  5. true -- incoming subcategory, false -- Expense subcategory
Record contains following fields:
  1. 2
  2. Record id, any string that can uniquely identify the record.
  3. Parent subcategory id
  4. Eecord date(year-month-day hour:minute), ex. 2022-01-12 05:15
  5. Amount, ex. 100.00
  6. 0
  7. Merchant
  8. Detail
  9. Label id
labels contains following fields:
  1. 3
  2. Label id，any string that can uniquely identify the label.
  3. Label name
  4. Label color：RGB color value ranged from 0 to 1, seperated by |， ex. "0.5|0|0.5"
