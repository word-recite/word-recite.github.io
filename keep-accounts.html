1. Category and Subcategory

This application groups account records into subcategories, and group subcategories into categores. A set of categories and subcategories will be generated when this app is used at the first time. You can add/delete/modify these categories and subcategories for your requirement.

Notice:

  • To add category, enter 'Category' tab，click in the upper right corner.
  • To add subcategory, first enter one of the category pages by clicking the category. then clickin the upper right corner. The subcategory can be created as an incoming subcategory which only contains incoming records, or an expense subcategory which only contains expense records.
  • To edit/delete a category, long press the name of the category, select 'Edit' or 'Delete' in the popup menu. To edit/delete a subcategory, enter the category which contains the subcategory, long press the name of the subcategory and select corresponding item in the popup menu.

2. Book keeping

Enter 'Category' or 'Summary' page, click "Record" button. filling information of the new record, click "Add" in the upper right corner.

Notice:

  • Enter "Settings" tab, select "Auto Record" in the upper section. Here, you can config the app to add record automatically in a specific day of each month (For example, add X dollar expense at day 31 of each month for iCloud monthly service fee). If a date is not valid in certain month, the auto-created record will be set for the last day of the month. (February does not have day 31, so the new record date will be set to 29 February or 28 February)
  • When adding the record, you can add picture from the Photos or Camera in the Image tab(For example, add the image of the receipt or invoice). Slide finger on the image will show a green rectangle on the image. Click "Crop" button in the bottom, will zoom in and crop the image.

3. Summary

Enter "Summary" tab, or enter "Category" tab then navigate to a subcategory page. Here you can view all the records in a particular day. The total expense and income of the whole year and month will also be shown. Click the year, month and day text on the top will pop up menu to switch displaying date.

Notice:

  • The popup menu for switching date will only contain date which has records.
  • You can select "All" to show records of the whole year(only valid in subcategory pages) or month.
  • You can edit/delete record by long pressing the record.
  • You can move the records to different subcategory. First click in the upper-right corner and select "Start Selection" in the popup menu. Then select records which need to be moved. Enter the target subcategory page, click in the upper-right corner and select "Move to here". Click "Done selection" in the same popup menu to finish selection state.

4. Chart

In "Chart" tab, you can view statistic on the records by different angle. For example, show total sum of the records for each year, then in each year, show total sum of the records for each month. By click "Reset" in the upper right corner of "Summary fields and order" section, the following fields and order will be set for histogram:
  • Balance Summary(This field can not be deleted and moved)
  • Yearly Summary
  • Monthly Summary
  • Daily Summary
  • Summary By Category
  • Summary By Subcategory
  • Summary By Merchant
By click "Edit" in the upper right corner of "Summary fields and order" section, you can delete or reorder these fields.

5. Settings

Enter the “Settings” tab, here you can use following functions and settings.

Upgrade App
(unrestricted use) 		Before paying for upgrading the app, you can only add 50 records and "Import from file" function can not be used. Click"Upgrade App at price XXX" to pay and upgrade the app. After upgrade, text “Purchased” will replace the upgrade button.

Notice:
If you reinstalled the app, or installed the app on another device after upgrading, you can click "Restore Purchase" to restore purchase state.
Sync iCloud Sync categories, subcategories and records to iCloud. The local updates will be uploaded to iCloud, and the remote changes will be downloaded to local device. Note that if you have updates on same field of the same category or subcategory or record from both local and iCloud, the local update will overwrite iCloud update.
Export to file Export categories, subcategories and records data to CSV file. Note that image data in records will be ignored.
Import from file Import categories, subcategories and records data from CSV file. Note that only data in CSV file that does not exists in local device will be imported.
